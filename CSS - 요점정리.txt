★CSS - 웹페이지 기획(디자인)

★CSS를 삽입하는 방법
＊Style태그 사용 - <style></style>
＊Style속성 사용 - style=""

★STYLE
[스타일 태그] = <style></style>
ㄴ태그 사용 - <style> a { 제원값; }
ㄴex)<style> a { color:red; }
ㄴ해당 태그에 해당되는 태그의 모든 색깔이 빨간색으로    바뀌는 것임.
ㄴ{제원값;} = 효과(Declaration)
ㄴex){효과(color:red;)}
ㄴ속성값(제원값) = Property(속성) Value(값)
ㄴex)color:red;

[스타일 속성] = <Tag style="제원값"></tag>
ㄴ태그 선택자 사용 안해도 됨 - style=""
ㄴex)<a style="color:red">
ㄴ해당 태그안에 스타일 속성을 부여하여 해당 태그의 모든   색깔이 빨간색으로 바뀌는 것임.

★속성(Property)
[Property - 속성(제원)]
1.color : 색깔
ㄴValue = red, black, yellow, etc.
2.font-size : 글자 크기
ㄴValue = (N)px, small~large, 10%~1000%, etc.
3.text-align : 텍스트 정렬
ㄴValue = center, left, right, justify, etc.
4.display : 요소 크기를 결정
ㄴValue = block, inline, none, etc.
5.border : 테두리
6.text-decoration : 밑줄
ㄴ Npx, none, etc.
7.min-height/width/etc. : 최소 세로값/가로값/etc
8.max-height/width/etc. : 최대 세로값/가로값/etc

★태그 선택자
[class - 태그 선택자]
ㄴclass 선택자는 중복 사용이 가능하다.
ㄴ하나의 클래스의 여러개의 값을 부여할 수 있으며 여러 개의 선택자를 생성하여 하나의 태그를 공동으로 제어할 수 있음.
ㄴclass라는 하나의 그룹에 들어가있는 속성들에게 모두 효과를 부여함.
ㄴclass를 대체하는 기호는 점(.)임.
ㄴ예를 들어 clsss 안에 saw라는 속성에게 효과를 주고 싶다면 ex) .saw가 됨.
ㄴ최근에 사용한 명령이 전에 사용한 명령보다 강한 영향력을 가짐.

[id - 태그 선택자]
※id 선택자는 단 하나만 있는 고유한 값 즉,주민등록증 같은 선택자이기 때문에 중복 사용이 불가능하다.
ㄴclass 선택자보다 우선순위가 높음.
ㄴid를 대체하는 기호는 샵(#)임.
ㄴid라는 하나의 그룹에 들어가있는 속성들에게 모두 효과를 부여함.
ㄴclass선택자보다 우선순위가 높기 때문에 class선택자를 나중에 사용해도 id선택자가 먼저 사용되게 됨.
ㄴ예를 들어 class안에 active가 있고 active보다 id 안에 saw라는 속성에게 우선순위를 주고 싶다면 ex) #saw가 됨.

★태그 선택자 정리
기본 태그 선택자 - a태그,h태그 등 기본 태그들을 스타일을 이용하여 효과를 부여하는 선택자
ㄴ모든 선택자들중 제일 약하다.

class 선택자 - 태그안에 있는 class속성 안의 모든 속성값에 효과를 부여하는 선택자
ㄴ기본 태그 선택자보다는 강하고 id 선택자보다는 약하다.

id 선택자 - 태그안에 있는 id속성 안의 모든 속성값에 효과를 부여하는 선택자
ㄴ모든 선택자중 제일 강하다.
ㄴ고유한 하나의 값이기 때문에 항상 우선순위가 제일 높다.

★div태그, span태그
디자인의 용도로만 사용하는 태그
div = block Element
span = inline Element

★박스모델
스타일 태그안에서 선택자를 사용하여 CSS스타일을 부여했을 때 해당 웹페이지에 나타나는 박스형태의 영역을
박스모델이라 하며, 해당 영역을 다른 속성(width, height, border 등등)을 사용하여 박스의 영역을 사용자가
원하는대로 바꿀 수 있다.

★그리드
두 개의 문단을 나란히 정렬하게 만들려고 할 때 사용하는 CSS 레이아웃
ㄴgrid라는 id선택자를 만들어 style 태그 안의 속성으로 제원값을 줘 그리드를 사용할 수 있다.
ㄴ태그안의 문장이 늘어날수록 같은 태그안의 다른 문장의 태그 영역도 같이 증가한다.

★미디어 쿼리
＊반응형 디자인
ㄴ미디어 쿼리 코드 : @media
웹페이지의 요소를 정해진 크기에 따라 반응형(보였다 안보였다)으로 만들어주는 CSS 레이아웃
ㄴ해당 미리어 쿼리를 사용하려고 할 때 예시
#1=>웹페이지의 요소를 최소800px이상일 때 none으로 지정할 수 있게 설정하고싶다.
/＊@media(min-width: 800px)
   div{
    display:none;
   }
*/
#2=>웹페이지의 요소를 최대800px 이하 값들만 none으로 지정할 수 있게 설정하고싶다.
/＊@media(max-width: 800px)
   div{
    display:none;
   }
*/

★링크
＊중복 제거 / CSS 스타일 압축
ㄴCSS 링크 코드 : <link rel="stylesheet" href="파일명"
스타일 태그 안에 사용되는 모든 스타일 속성들을 중복으로 계속 붙여넣는 형식으로 사용할 수 있지만 해당 스타일을 적용시킬 웹페이지가 100개 이상일 경우 스타일을 모두 일일이 붙여넣는 형식으로 적용시키기엔 한계가 있다. 그러므로 하나의 스타일이 적용된 CSS파일을 하나 생성하여 그 CSS파일에 원하는 스타일 속성들을 모두 적은 뒤 CSS파일로 만든 다음 CSS 링크를 사용하여 해당 CSS파일명을 입력해 해당 파일이 적용된 웹페이지들은 모두 스타일이 적용되게 된다.
